## Installation instruction's

### Windows
To start out go to the packages tab and install RSVP.exe
The app has no installer and instead runs from the file itself.

### Linux
Download the Executable Linux app file form the Releases tab
Launch the RSVPLinux app from your file manager

## Using Phant

To use Phant first launch the application and find an article online (Not all providers will work with Phant) past in the URL to the URL bar and press load, if the article is available it will load and you can read it

If the article cannot be reached or is incompatible you will get an error most likely saying
